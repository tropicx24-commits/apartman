# Apartman Yönetim Uygulaması

Apartman yönetimi ve aidatlandırma işlemleri için geliştirilmiş Python tabanlı masaüstü uygulamasıdır.

## Özellikler

- 🏢 Apartman ve daire yönetimi
- 💰 Aidat takibi ve yönetimi
- 📊 Raporlama ve analiz
- 👥 Sakin yönetimi
- 📋 Ödemeler ve borç takibi
- 🔍 Veri sorgulama ve filtreleme

## Teknoloji

- **Dil:** Python 3.x
- **Arayüz:** PyQt5 / Tkinter (GUI)
- **Veritabanı:** SQLite / Excel

## Kurulum

### Gereksinimler

```bash
Python 3.7+
```

### Bağımlılıkları Yükleme

```bash
pip install -r requirements.txt
```

## Kullanım

Uygulamayı çalıştırmak için:

```bash
python apartman_aidat_app_v6_entegre_duzeltilmis_final_fix3.py
```
